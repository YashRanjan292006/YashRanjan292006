### 🚀 Flagship Architecture & Systems

#### 1. Smart Knowledge Vault | *AI Document Intelligence Platform*
An industry-standard AI-powered RAG Document Intelligence Platform designed to process, embed, and query complex enterprise documents securely.
- **Tech Stack:** React, Node.js, Express, MongoDB Atlas Vector Search, Redis, Gemini 1.5 Flash, `@xenova/transformers`.
- **Key Engineering Decisions:**
  - **Local Semantic Embeddings:** Deployed `@xenova/transformers` locally to reduce cloud dependency and guarantee millisecond retrieval times.
  - **Advanced RAG Pipeline:** Integrated Gemini 1.5 Flash for high-context natural language synthesis based on strict chunk boundaries.
  - **Infrastructure Resilience:** Implemented Redis caching layers and aggressive rate-limiting safeguards to ensure high availability.
  - **UI/UX Strategy:** Developed a modern, glassmorphism-based UI interface for smooth document interaction.
